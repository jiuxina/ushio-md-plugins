# GitHub 风格提示框插件

快速插入 GitHub 风格的 NOTE/TIP/IMPORTANT/WARNING/CAUTION 提示框，让你的 Markdown 文档更加清晰易读。

## 功能介绍

此插件在编辑器工具栏中添加 5 个快捷按钮，可一键插入以下提示框：

| 按钮 | 类型 | 效果 |
|------|------|------|
| ℹ️ | NOTE | 蓝色提示框，用于补充说明 |
| 💡 | TIP | 绿色提示框，用于技巧建议 |
| ⭐ | IMPORTANT | 紫色提示框，用于重要信息 |
| ⚠️ | WARNING | 黄色提示框，用于警告信息 |
| ❌ | CAUTION | 红色提示框，用于危险警告 |

## 使用方法

1. 在编辑器中将光标放置在需要插入提示框的位置
2. 点击工具栏中对应的提示框按钮
3. 在插入的提示框内输入你的内容

## 示例

在 Markdown 中输入：

```markdown
> [!NOTE]
> 这是一条补充说明信息
```

渲染效果就会是一个蓝色的提示框。

## 兼容性

- 最低版本要求：ushio-MD 1.2.0
- GitHub、GitLab 等平台原生支持此语法渲染
